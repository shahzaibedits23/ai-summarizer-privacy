Privacy Policy for AI Web Page Summarizer Last Updated: February 2025 1. Introduction AI Web Page Summarizer ("we," "us," "our," or "Extension") is committed to protecting your privacy. This Privacy Policy explains how our Chrome extension handles your information. 2. Information We Collect Our extension collects minimal data, all stored locally on your device: Extension Settings: Your preferences (summary length, theme, etc.) stored in browser local storage Usage Statistics: Number of summaries created (for free tier tracking) API Keys: If you provide your own Anthropic API key, it's stored locally in your browser We do NOT collect: Your browsing history Personal information or identity data Tracking cookies Analytics or telemetry Page content (except when you explicitly request summarization) 3. How We Use Your Data Your data is used only for: Providing summarization services when you request them Storing your preferences and settings Tracking your usage against the free tier limit 4. Data Processing Page Content: When you request a summary: The page content is extracted locally in your browser Text is sent to Anthropic's Claude API for processing The summary is returned and displayed in the extension Content is NOT stored on our servers Anthropic may process this data according to their terms Your Settings: All extension settings are stored locally in your browser using Chrome's storage API. They never leave your computer. 5. Third-Party Services We use Anthropic's Claude API for AI summarization. When you request a summary: Text is sent to Anthropic servers Anthropic processes the text per their terms of service You should review Anthropic's Privacy Policy at https://www.anthropic.com/privacy If you use your own API key, you are responsible for managing it 6. Data Storage and Security All settings are stored locally on your device API keys are stored locally in your browser No data is transmitted to our servers We recommend using HTTPS and keeping your browser updated Chrome's storage system provides reasonable security for local data 7. Your Rights You have the right to: Access all data stored by the extension (visible in Chrome settings) Delete all extension data (via Chrome settings or extension removal) Modify your settings at any time Disable or uninstall the extension To clear all extension data: Go to chrome://extensions Find "AI Web Page Summarizer" Click "Clear data" or uninstall 8. Permissions Explained The extension requires these Chrome permissions: activeTab - To access the current page's title and metadata scripting - To extract text content from the page storage - To save your settings locally - To work on any website 9. Changes to This Policy We may update this policy occasionally. Changes will be noted with an updated "Last Updated" date. Continued use of the extension constitutes acceptance of changes. 10. Compliance This extension complies with: Chrome Web Store Developer Program Policies Chrome Extension Manifest V3 requirements GDPR (for EU users - we don't collect personal data) CCPA (for California residents - minimal data collection) 12. Data Deletion To completely remove all data: Uninstall the extension from Chrome All local data is automatically deleted No data remains on our servers (we have none) Thank you for using AI Web Page Summarizer!
